on AWS

Tools setup on Ec2 instance:
----------------------------
ON CLI command:
---------------
1. Create Ec2 instance and add admin role

Launch new Ubuntu VM using AWS Ec2 ( t2.micro )

Connect to machine and install kubectl using below commands
curl -o kubectl https://amazon-eks.s3.us-west-2.amazonaws.com/1.19.6/2021-01-05/bin/linux/amd64/kubectl
chmod +x ./kubectl
sudo mv ./kubectl /usr/local/bin
kubectl version --short --client

Install AWS CLI latest version using below commands
sudo apt install unzip
curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
unzip awscliv2.zip
sudo ./aws/install
aws --version

Install eksctl using below commands
curl --silent --location "https://github.com/weaveworks/eksctl/releases/latest/download/eksctl_$(uname -s)_amd64.tar.gz" | tar xz -C /tmp
sudo mv /tmp/eksctl /usr/local/bin
eksctl version


2. EKS setup by client ec2 instance (Paid service)

2.1 Crate cluser:

eksctl create cluster --name dk-cluster-1 --region us-east-1 --node-type t2.medium  --zones us-east-1a,us-east-1b

kubectl get nodes
kubectl get pods
